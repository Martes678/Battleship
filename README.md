# Hundir la flota
Implementación del juego hundir la flota en C++.
Como se puede apreciar en la carpeta _Hundir_ _la_ _flota_, se pueden ver las clases que hacen posible que el juego funcione correctamente.
El archivo que gestiona todas las clases es _Hundir la flota.cpp_.
A parte del código, hay un manual de usuario en la carpeta principal.
